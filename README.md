# 🎓 نظام إدارة المدرسة

## 📌 نبذة عن المشروع

نظام إدارة مدرسة متكامل تم تطويره باستخدام **Python و Django** بهدف توفير حل إلكتروني شامل لإدارة العمليات التعليمية والإدارية داخل المدرسة.

يساعد النظام على تنظيم وإدارة بيانات الطلاب والمعلمين والفصول والمواد الدراسية والحضور والغياب والاختبارات والدرجات من خلال منصة واحدة سهلة الاستخدام.

تم تصميم المشروع لتقليل الأعمال الورقية، تسريع الوصول إلى المعلومات، وتحسين كفاءة الإدارة المدرسية، مع إمكانية تطويره مستقبلًا وإضافة المزيد من الخدمات.

---

# ✨ مميزات النظام

## 👨‍🎓 إدارة الطلاب
- إضافة وتعديل وحذف بيانات الطلاب.
- إنشاء ملفات خاصة بالطلاب.
- البحث عن الطلاب وإدارة معلوماتهم.
- متابعة السجلات الأكاديمية.

## 👨‍🏫 إدارة المعلمين
- إضافة وإدارة بيانات المعلمين.
- تنظيم معلومات المعلمين.
- ربط المعلمين بالفصول والمواد.

## 🏫 إدارة الفصول والمواد
- إنشاء الصفوف الدراسية.
- إدارة الشعب والفصول.
- تنظيم المواد الدراسية.

## 📝 نظام الحضور والغياب
- تسجيل حضور الطلاب يوميًا.
- متابعة سجلات الغياب.
- إعداد تقارير الحضور.

## 📊 الاختبارات والدرجات
- إضافة درجات الاختبارات.
- متابعة أداء الطلاب.
- تنظيم النتائج الأكاديمية.

## 🔐 المستخدمون والصلاحيات
- نظام تسجيل دخول آمن.
- إدارة المستخدمين.
- التحكم في صلاحيات الوصول.

## 📈 لوحة التحكم
- عرض إحصائيات المدرسة.
- متابعة أعداد الطلاب والمعلمين.
- إدارة العمليات من خلال لوحة تحكم سهلة.
